Dark Mode

The darkmode on this site is pure CSS as well. And so far, I could even avoid media queries.

This is thanks to system-color variables which you can use wherever you'd otherwise define a fixed color (using rgb(), hsl() and so on). Then the browser dynamically supplies colors of the current system appearance.

When the system appearance is set to light, Canvas will return white, whereas CanvasText black. In dark mode, it is approximately the other way around.
